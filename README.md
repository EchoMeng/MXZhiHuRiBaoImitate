# MXZhiHuRiBaoImitate
根据https://github.com/izzyleung/ZhihuDailyPurify/wiki/%E7%9F%A5%E4%B9%8E%E6%97%A5%E6%8A%A5-API-%E5%88%86%E6%9E%90 提供的API制作的iOS客户端

实现的功能： 

1 主页面上拉下拉刷新； 

2 点击cell显示新闻详情； 

3 新闻详情页左右滑动显示上/下篇内容； 

4 左滑抽屉效果； 

5 分类信息栏点击显示相应消息列表； 

6 登录界面搭建。
